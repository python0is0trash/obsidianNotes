Используется для **анализа данных из Amcache.hve** - это файл реестра Windows, который содержит информацию об установленных программах, исполняемых файлах и других артефактах, связанных с приложениями. Этот файл используется Windows для ускорения запуска программ и улучшения производительности.
## Пример вывода
___
```bash
EntryType  Path                              Company       LastModifyTime   LastModifyTime2   InstallTime  CompileTime   SHA1     Service     ProductName     ProductVersion

File       C:\Program Files\example.exe      Example Corp  time UTC         time UTC          time UTC     time UTC      \hash1\  N/A         Example App     1.0.0
File       C:\Windows\System32\notepad.exe   Microsoft     time UTC         time UTC          N/A          time UTC      \hash2\  N/A         Notepad         10.0.19041
```
## Что означает вывод
___
1. **EntryType:** Тип записи. Например, может указывать на то, является ли запись файлом, программой или другим типом объекта.
2. **Path:** Полный путь к файлу на диске. Это может быть путь к исполняемому файлу (например, .exe) или другому файлу, зарегистрированному в Amcache.
3. **Company:** Название компании-разработчика программы. Это поле часто заполняется для исполняемых файлов, которые имеют информацию о производителе в своих метаданных.
4. **LastModifyTime:** Время последнего изменения файла.
5. **LastModifyTime2:** Дополнительное поле, связанное с временем изменения. В некоторых случаях это может дублировать LastModifyTime или содержать другую связанную временную метку.
6. **InstallTime:** Время установки программы.
7. **CompileTime:** Время компиляции исполняемого файла.
8. **SHA1:** Хэш-сумма файла по алгоритму SHA-1. 
9. **Service:** Указывает, связана ли запись с сервисом Windows. 
10. **ProductName:** Название продукта (программы). Это поле часто заполняется для установленных программ.
11. **ProductVersion:** Версия продукта (программы).
